# 🌤️ SkyShop - E-Commerce Marketplace

SkyShop is a full-stack e-commerce platform built with **Python, Django, and Django REST Framework (DRF)**, inspired by Amazon's core marketplace functionality. It provides a secure and scalable shopping experience for both customers and sellers.

## 🚀 Features

* 🔐 JWT Authentication, Google OAuth, Password Recovery
* 👥 Role-Based Access (Customer & Seller)
* 🛍️ Product Management with Multi-Image Uploads
* 🔎 Product Search & Filtering (Category, Brand, Price)
* 🛒 Shopping Cart & Order Management
* 📦 Order Tracking (Ordered → Packed → Shipped → Delivered)
* ⭐ Reviews & Ratings System
* ❤️ Wishlist & Coupon Support
* 💳 Razorpay / Stripe Payment Integration
* 📄 Automated PDF Invoice Generation
* ☁️ Cloudinary Image Storage
* 📧 Email Notifications for OTPs and Orders
* 📊 Admin Dashboard with Revenue, Orders, Products, and Users Analytics

## 🛠️ Tech Stack

* **Backend:** Python, Django, Django REST Framework
* **Authentication:** JWT, Google OAuth
* **Database:** PostgreSQL / MySQL
* **Payments:** Razorpay, Stripe
* **Cloud Storage:** Cloudinary
